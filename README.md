# Mindmap System (V1)

This repository contains a no-code friendly, system-ready mind map built using
Markdown files.

Primary goals:
- Visual, expandable thinking for non-coders
- Git-versioned knowledge
- Future-ready for automation, parsing, and visualization

V1 Scope:
- Visualization via Markdown + Obsidian
- Editing without code
- Stable structure for future code layers

Do not change structure without updating rules.
